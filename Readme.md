Annotex is a simple web-based annotation tool that allows users to highlight text and attach notes to selected content.  
The project focuses on clean UI and core JavaScript logic for text selection, annotation, and persistence.

---

#  Features

- Highlight selected text
- Add notes linked to highlighted text
- Click highlight -> jump to related note
- Click to note -> focus on highlighted text
- Edit notes anytime
- Persistent storage using browser LocalStorage
- Clean, dark-themed UI

---

#  Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)
- LocalStorage (for saving annotations)

---

#  How to Run the Project

1. Clone the repository or download ZIP
2. Open the project folder
3. Open `mainPage.html` in a browser  
   *(or use Live Server in VS Code)*
